# elizax_talenx

talenx HR·성과관리 SaaS 사용자 앱 화면 복제 목업 (단일 HTML).

실제 서비스(app.talenx.com) 라이브 대조로 재구축한 정적 목업입니다. 외부 의존성 없이 `index.html` 하나로 동작합니다.

## 구성

- **홈 대시보드** + GNB 8개 메뉴
  - 업무관리 (업무보드 · 스크럼보드)
  - 성과관리 (목표 현황 · 피드백 · 1:1 미팅 · 리뷰)
  - 360 진단 (360 피드백 현황)
  - 평가관리 (평가 매트릭스 · 탈렌트 세션)
  - 급여관리 (내 급여 · 연말정산)
  - 근무관리 (내 근무 · 내 휴가 · 구성원 근무 · 구성원 휴가 · 근무스케줄 · 위치정보 제공 조회 · 연차촉진)
  - 인사관리 (사용자 정보 · 구성원 정보 · 인재 검색 · 인원 현황)
  - 승인/결재 (받은 문서 · 보낸 문서 · 서명 문서)
- GNB 라우팅 + 각 메뉴 좌측 서브내비 전환 동작 (순수 JS, 스코프 IIFE)
- 디자인 토큰: Primary `#1F7AF0` / Pretendard

## 인터랙션 (people.talenx.com 대조 완전 동작)

- **모든 버튼/탭/케밥(⋮)/필터/행/더보기가 클릭 시 실제 동작** — 스크린샷 대응 모달·드로어·컨텍스트메뉴·펼침·토스트. 죽은 클릭 0개.
- 공통 헤더 3종: 통합 검색 모달(⌘/Ctrl+K), 알림 패널(승인필요·읽지않음·전체), 프로필 드로어(마이페이지/계정설정/고객센터/관리자메뉴/로그아웃).
- 인사관리 "내 정보" 13개 탭 전부 실제 콘텐츠. 각 메뉴 상세/작성/신청/필터 화면을 모달·드로어로 재현(데이터는 목업, 서버 저장 없음).
- 전역 헬퍼 `window.tx`(modal/drawer/toast/confirm/cmenu)로 통일.
