# elizax_talenx

talenx HR·성과관리 SaaS 사용자 앱 화면 복제 목업 (단일 HTML).

실제 서비스(app.talenx.com) 라이브 대조로 재구축한 정적 목업입니다. 외부 의존성 없이 `index.html` 하나로 동작합니다.

## 구성

- **홈 대시보드** + GNB 8개 메뉴
  - 업무관리 (업무보드 · 스크럼보드)
  - 성과관리 (목표 현황 · 피드백 · 1:1 미팅 · 리뷰)
  - 360 진단 (360 피드백 현황)
  - 평가관리 (평가 매트릭스 · 탈렌트 세션)
  - 급여관리 (내 급여 · 연말정산)
  - 근무관리 (내 근무 · 내 휴가 · 구성원 근무 · 구성원 휴가 · 근무스케줄 · 위치정보 제공 조회 · 연차촉진)
  - 인사관리 (사용자 정보 · 구성원 정보 · 인재 검색 · 인원 현황)
  - 승인/결재 (받은 문서 · 보낸 문서 · 서명 문서)
- GNB 라우팅 + 각 메뉴 좌측 서브내비 전환 동작 (순수 JS, 스코프 IIFE)
- 디자인 토큰: Primary `#1F7AF0` / Pretendard
