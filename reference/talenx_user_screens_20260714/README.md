## 파일 명명 규칙

- `<번호>_<영문키>.jpg` = 해당 화면 뷰포트
- `..._full.jpg` = 전체 스크롤(페이지 전체 높이) 캡쳐
- `..._t0 / _t1 / ...` = 해당 화면 내 **탭/필터 전환** 상태별 캡쳐
  - 예: `10_hrm_my_info_t0~t12` = 내 정보 13개 탭(인사정보/개인정보/학력경력/자격어학/병역/장애보훈/포상징계/평가/노동조합/교육/급여/인사신청/제증명)
  - 예: `20_att_work_t0~t2` = 실근무/소정근무/잔여근무
  - 예: `80_wf_received_t0~t2` = 진행중/중요/완료
