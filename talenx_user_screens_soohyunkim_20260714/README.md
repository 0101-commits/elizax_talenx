## 잘림/blank 방지 처리 (이번 요청 핵심)

- 로그인 직후 **"비밀번호 변경 필요" 팝업**(관리자 지정 임시 비번, swal2 모달)이 모든 화면을 덮고 클릭을 가로챔
  → 매 페이지 진입 시 **"다음에 변경"** 으로 닫아 화면 가림/탭 클릭 실패 제거 (비번은 변경하지 않음)
  → 해당 팝업 자체도 `00_entry/01b_popup_pw_change.jpg` 로 기록
- 페이지별 **networkidle 대기 + 단계 스크롤(lazy 렌더 강제) + 재대기** 후 촬영
- 촬영물 전수 검증: 본문 텍스트량 + 파일 크기로 blank 판정, 미달 시 재촬영. **blank/잘린 화면 0건**

## 권한 차이 (1차 계정과 비교)

이 계정(미주팀)은 인사 관리자 메뉴 일부 **접근 불가** → 캡쳐 대상에서 제외:

| 경로 | 결과 |
|---|---|
| `/hrm/employee_search` (인재 검색) | 404 / 메뉴 없음 → 제외 |
| `/hrm/statistics` (인원 현황) | 404 / 메뉴 없음 → 제외 |

- 인사관리 좌측 메뉴 = **내 정보 · 구성원 정보** (2개)
- 성과관리 좌측 메뉴 = 목표 · 피드백 · 1:1 미팅 · 리뷰
- 평가관리 좌측 메뉴 = 평가 · 탈렌트세션

| `30_업무관리` | 업무관리 | 업무보드(전체/즐겨찾기), 스크럼보드 |

## 파일 명명 규칙

- `<번호>_<영문키>.jpg` = 화면 뷰포트 / `..._full.jpg` = 전체 스크롤 캡쳐
- `..._t0 / _t1 / ...` = 화면 내 탭·필터 전환 상태별 캡쳐
  - `10_hrm_my_info_t0~t12` = 내 정보 13개 탭
  - `20_att_work_t0~t2` = 실근무/소정근무/잔여근무
  - `80_wf_received_t0~t2` = 진행중/중요/완료
