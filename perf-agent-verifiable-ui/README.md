# 성과관리 AI Agent — Verifiable Answer UI

성과관리/평가 E2E AI Agent를 **Verifiable Answer UI 4대 원칙**에 따라 재설계한 인터랙티브 목업.
`index.html` 하나로 완결 — 브라우저로 바로 열면 됩니다. (GitHub Pages 배포 가능)

## 4대 시그널 레이어 (answer receipt)

모든 답변을 "검증 가능한 객체(answer receipt)"로 렌더링한다. 주황색 번호 = 레이어 위치.

1. **기준 시점 칩** — bitemporal as-of 셀렉터. 과거 시점(평가 시즌·마감 스냅샷) 기준 재조회. *(칩 클릭 시 스냅샷 전환)*
2. **계산·근거 트레이스** — 규칙 버전 배지, 규정 칩, 원천 레코드 식별자(`goal.2026H1.kr3`) 펼침.
3. **감사 가능 배지** — 감사 로그 기록 여부 + 권한 내 탐색 범위.
4. **What-if 재계산** — 계산 엔진 실증. **버튼이 실제로 동작**한다.

## 3대 UX 형태 (하나의 에이전트, 화면만 바뀜)

기본은 ①, 필요 시 ②·③으로 승격. 3모드 모두 **확인 내역 · 근거 인용 · 승인/수정/보류 게이트**를 공유.

| 형태 | 태그 | 노출 | 시연 |
|---|---|---|---|
| ① 도킹 대화창 | 기본·상주 | talenx 화면 우측 340px 상주, 등급 초안 영수증 | as-of 전환, 트레이스 토글, What-if 재계산 |
| ② 선제 팝업 | 보조·선제 | 우하단 앰버 카드, 신호 배치 감지 | — |
| ③ 전체화면 딥워크 | 심층·전환 | **W2 목표 설정 화면 재현** — 좌 캔버스(작업 표면) + 우 패널(근거 스트림) | **▶ 목표 초안 실행** → 노드가 채워지는 라이브 시뮬 |

전체화면 = 성과관리/평가 E2E App 목표 설정 기능 화면: 6대 UI 컴포넌트(글로벌 바·내비·중앙 캔버스·컨텍스트 패널·명령어 입력창·상태 바), 5단계 스텝퍼(지침 수립→자기 목표→검토 회의→피드백→목표 확정), 3종 Agent 병렬 노드 캔버스.
